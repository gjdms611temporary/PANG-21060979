# PANG (팡) 게임 — Mission 1 PRD

## 1. 개요
PANG(팡)은 1989년 Mitchell Corporation이 출시한 아케이드 게임(북미명: Buster Bros.)으로,
화면 위를 통통 튀어 다니는 풍선(볼)을 무기로 터뜨려 스테이지를 클리어하는 액션 게임이다.
Mission 1은 이 시리즈의 가장 첫 번째 스테이지로, 게임의 기본 규칙을 플레이어에게 익히게 하는
튜토리얼 성격의 단순한 구성을 가진다.

## 2. 핵심 목표
- 화면에 등장하는 모든 풍선(볼)을 제거하면 스테이지(Mission) 클리어
- 풍선에 몸이 닿으면 즉시 사망(목숨 1개 소모), 스테이지 재시작
- 정해진 목숨을 모두 소진하면 게임 오버

## 3. 기본 게임 규칙
- **플레이어 캐릭터**: 화면 하단에서 좌우로만 이동 가능
- **기본 무기**: 수직으로 발사되는 와이어(Wire) 1발
  - 기본 상태에서는 화면에 한 발만 존재 가능(발사 후 벽/천장에 닿거나 풍선에 맞아야 재발사 가능)
- **풍선 분열 규칙**
  - 가장 큰 풍선을 맞히면 같은 크기의 작은 풍선 2개로 분열
  - 작은 풍선은 Wire를 맞으면 소멸
  - 풍선은 좌/우 방향으로 공처럼 튀겨진다.
  - 중력의 영향을 받고, 튕길 때마다 같은 높이까지 올라간다.
  - 화면 밖으로 풍선이 나갈 수 없다.
- **스테이지 클리어 조건**: 화면 내 모든 풍선(분열체 포함)을 제거

## 4. Mission 1의 특징
- 시리즈 첫 스테이지답게 등장 풍선 수가 적고 크기도 상대적으로 작아 난이도가 낮음
- 특수 아이템이나 보조 무기(더블 작살, 발톱형 와이어 등) 없이 기본 작살만으로 클리어 가능한 구성
- 배경은 시리즈 초반 테마(도시/랜드마크)로, 이후 스테이지로 갈수록 배경이 세계 각지로 변경되는 구조의 시작점 역할
- 플레이어가 이동/발사/충돌 회피라는 3가지 기본 조작만으로 학습 가능하도록 설계됨

## 5. 점수 및 보너스 요소
- 풍선을 터뜨릴 때마다 점수 획득(작은 풍선일수록 처치 시 점수가 높은 경향)
- 스테이지 진행 중 과일 등 보너스 아이템이 등장하며, 주워 먹으면 추가 점수 획득
- 일정 점수 도달 시 목숨(1UP) 추가 지급

## 6. 실패 조건
- 풍선(어떤 크기든)에 플레이어 캐릭터가 닿으면 사망
- 목숨을 모두 소진하면 게임 오버(이어하기 미지원 시)

## 7. 비고
본 PRD는 세부 수치(속도, 점수 배점, 아이템 확률 등)를 포함하지 않고,
Mission 1의 전반적인 게임플레이 컨셉과 규칙을 정리한 개요 수준의 문서이다.
세부 스펙은 후속 상세 기획 문서에서 확정한다.
