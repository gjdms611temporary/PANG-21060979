# Mission 1 난이도 규칙

공통 게임 룰은 [game_rule.md](game_rule.md)를 따르며, 본 문서는 Mission 1에 한정된 난이도/구성 규칙을 정의한다.

## 1. 목적
- 시리즈의 첫 스테이지로, 플레이어가 이동 / 발사 / 충돌 회피라는 3가지 기본 조작을 자연스럽게 익히도록 하는 튜토리얼 성격의 스테이지

## 2. 풍선 구성
- 등장 풍선 수: 소수(예: 대(大) 풍선 1~2개 수준, 실제 수치는 구현 단계에서 확정)
- 등장 풍선 크기: 대(大) 풍선까지만 등장(더 큰 초대형 풍선 없음)
- 분열 단계: 대 → 중 → 소 (총 3단계, [game_rule.md](game_rule.md) 3.1절과 동일)

## 3. 이동 속도
- 풍선의 좌/우 이동 속도 및 낙하 속도는 시리즈 내 다른 미션 대비 가장 느린 수준으로 설정
- 풍선이 튕기는 높이(바운스 높이)도 상대적으로 낮게 설정하여 플레이어의 반응 여유를 확보

## 4. 아이템 / 보조 무기
- Mission 1에서는 보너스 아이템(과일 등) 및 보조 무기(더블 와이어 등) 없이 기본 Wire 1발만으로 클리어 가능하도록 구성
- 특수 기믹(장애물, 이동 발판 등) 없음

## 5. 배경 및 테마
- 배경은 시리즈 초반 테마(도시/랜드마크)를 사용하며, 이후 미션으로 갈수록 배경이 세계 각지로 변경되는 구조의 시작점 역할을 한다

## 6. 클리어 조건
- 화면 내 모든 풍선(분열체 포함)을 제거하면 Mission 1 클리어 → 다음 미션으로 진행(또는 데모 버전의 경우 결과 화면 표시)

## 7. 실패 조건
- [game_rule.md](game_rule.md) 6절과 동일: 풍선과 접촉 시 사망, 목숨 소진 시 게임 오버

## 8. 비고
- 본 문서는 개요 수준의 난이도 규칙만 정의하며, 정확한 수치(속도 값, 등장 풍선 개수/좌표, 점수 배점 등)는 후속 상세 기획 및 구현 단계에서 확정한다
