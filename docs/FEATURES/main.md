# 메인 화면 (Main Screen) 구성

## 1. 개요
게임 실행 시 가장 먼저 표시되는 화면으로, 플레이어가 게임을 시작하거나 부가 정보를 확인할 수 있는 진입점 역할을 한다.

## 2. 화면 구성 요소
- **타이틀 로고**: 화면 상단 중앙에 "PANG" 게임 로고 표시
- **메인 메뉴**: 화면 중앙에 세로로 배치된 메뉴 목록
  - `게임 시작 (Start)` — Mission 1로 진입
  - `조작 방법 (How to Play)` — 이동/발사 등 기본 조작 안내
  - `설정 (Options)` — 사운드 On/Off 등 (추후 확장)
- **배경**: Mission 1과 동일 계열의 도시/랜드마크 테마 정적 이미지 또는 애니메이션
- **하단 정보**: 버전 정보, 저작권/크레딧 표기(선택)

## 3. 인터랙션
- 방향키(↑/↓) 또는 마우스로 메뉴 항목 간 이동
- Enter 또는 클릭으로 메뉴 항목 선택
- `게임 시작` 선택 시 Mission 1 화면으로 전환
- 메뉴 진입/전환 시 간단한 하이라이트 효과(선택된 항목 강조)

## 4. 화면 전환 흐름
```
메인 화면 --(게임 시작 선택)--> Mission 1
메인 화면 --(조작 방법 선택)--> 조작 안내 화면 --(뒤로가기)--> 메인 화면
```

## 5. 비고
- 최초 버전은 메뉴 항목 중 `게임 시작`만 동작하고, 나머지 항목은 UI만 배치 후 추후 기능 연결
- 세부 레이아웃 수치(폰트 크기, 여백 등)는 디자인 확정 후 별도 문서에서 다룬다
