# 게임 룰 상세 (Game Rule)

PRD([docs/PRD.md](../PRD.md))에 정의된 개념을 기반으로, 실제 구현에 필요한 수준의 상세 규칙을 정리한다.

## 1. 플레이어
- 화면 하단에서 좌우로만 이동 가능(상하 이동 불가)
- 화면 좌/우 벽을 넘어갈 수 없음(화면 안에서만 이동)
- 풍선(어떤 크기든)과 몸이 닿으면 즉시 사망 → 목숨 1개 소모 → 스테이지 재시작

## 2. 무기 (Wire)
- 플레이어 위치에서 수직(위 방향)으로 발사되는 와이어(Wire) 1발
- 기본 상태에서는 화면에 와이어가 한 발만 존재 가능
  - 이미 발사된 와이어가 존재하면 재발사 불가
  - 와이어가 화면 상단(천장)에 닿거나 풍선에 맞아 소멸해야 재발사 가능
- 와이어가 풍선에 닿으면: 풍선은 분열 또는 소멸 처리, 와이어는 즉시 소멸

## 3. 풍선 (Balloon)
### 3.1 크기 및 분열
- 풍선 크기는 대(大) → 중(中) → 소(小) 3단계로 구분
- 가장 큰(대) 풍선을 Wire로 맞히면 같은 크기 단계 아래의 풍선 2개로 분열
- 가장 작은(소) 풍선을 Wire로 맞히면 분열 없이 소멸

### 3.2 이동 규칙
- 풍선은 좌/우 방향으로 공처럼 튀기며 이동한다(수평 이동 + 수직 바운스 결합)
- 중력의 영향을 받아 낙하하며, 바닥/장애물에 닿아 튕길 때마다 이전과 동일한 높이까지 다시 올라간다(감쇠 없음)
- 화면 좌/우 벽에 닿으면 진행 방향이 반전되어 반사됨
- 풍선은 화면 밖(좌/우/상/하)으로 벗어날 수 없다 — 화면 경계에서 항상 반사/차단 처리

### 3.3 분열 시 이동 방향
- 분열된 2개의 하위 풍선은 분열 시점의 위치에서 좌/우로 각각 반대 방향 진행을 시작한다

## 4. 스테이지 클리어 / 게임 오버 조건
- **클리어 조건**: 화면 내 모든 풍선(분열체 포함)을 제거하면 해당 미션 클리어
- **실패 조건**: 풍선과 접촉하여 목숨을 모두 소진하면 게임 오버(이어하기 미지원 시 메인 화면으로 복귀)

## 5. 점수 및 보너스
- 풍선을 Wire로 제거할 때마다 점수 획득
  - 풍선 크기가 작을수록 처치 시 획득 점수가 높음(대 < 중 < 소)
- 스테이지 진행 중 등장하는 과일 등 보너스 아이템 획득 시 추가 점수
- 누적 점수가 일정 기준에 도달하면 목숨(1UP) 1개 추가 지급
